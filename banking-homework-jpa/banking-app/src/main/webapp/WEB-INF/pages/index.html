Readme

URL HTTP method Request Body Action
Users
/user/all GET - Gets all users
/user/all/ GET - Gets all users
/user/{id}/ GET - Gets user by its id
/user/{id}/ DELETE - Deletes user by its id
/user/ POST user data: {"userId":integer, "name":string} Updates existing user
/user/ PUT user data: {"name":string} Creates new account
Accounts
/account/all GET - Gets all accounts
/account/all/ GET - Gets all accounts
/account/{id}/ GET - Gets account by its id
/account/{id}/ DELETE - Deletes account by its id
/account/ POST account data: {"accountId":integer, "userId":integer, "amount":double} Updates existing account
/account/ PUT account data: {"userId":integer, "amount":double} Creates new account
Service request
/flush_database GET - WARNING: removes all data from database!
